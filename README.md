# Tampermonkey Userscripts

Small collection of personal Tampermonkey userscripts.

## Scripts

### Toggle Chatgpt Temporary Chat Mode (Alt+T)
- Site: https://chatgpt.com
- Shortcut: Alt + T  
- Install:  
https://raw.githubusercontent.com/ARYAN-9099/tampermonkey-userscripts/main/toggle-temporary-chat-mode.user.js

### YouTube Progress Bar Back to Red
- Site: https://youtube.com  
- Install:  
https://raw.githubusercontent.com/ARYAN-9099/tampermonkey-userscripts/main/youtube-progress-bar-red.user.js
